# Syed Asif's Portfolio

A modern, interactive portfolio website showcasing full-stack development skills, projects, and professional experience.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Scrolling**: Enhanced user experience with smooth scroll behavior and smart anchor navigation with header offset
- **Popup Animations**: Interactive pop animations triggered on button clicks and element focus
- **Separated Sections**: Distinct Education and Experience sections for clear information organization
- **Education Cards**: Beautiful card-based layout for educational background and qualifications
- **Project Showcase**: Featured projects with live demo links
- **Skills Grid**: Technical skills displayed in an organized grid format
- **Contact Cards**: Easy access to email, GitHub, LinkedIn, and phone contact options
- **Modern UI**: Glassmorphism effects, gradient backgrounds, and smooth transitions

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structured content
- **CSS3**: Advanced styling with animations, gradients, and glassmorphism effects
- **JavaScript**: Event delegation, IntersectionObserver for scroll animations, smooth anchor scrolling
- **Font**: Poppins (Google Fonts)

## 🔧 Interactive Elements

### Button Animations
- All buttons (`btn`), navigation links, and cards trigger a pop animation on click
- Pop animation includes scale effect and glow shadow
- Smooth cleanup after animation completes

### Input Focus
- Input fields and elements with placeholders trigger pop animations on focus
- Hover effects on placeholder elements

### Scroll Behavior
- Smooth scrolling across the entire page
- Smart anchor navigation that accounts for fixed navigation header
- Momentum scrolling support on iOS

## 📝 Customization

### Colors
Edit the CSS variables in `portfolio.css`:
- Primary: `#00bfff` (cyan)
- Background: `linear-gradient(135deg, #031225 0%, #071836 33%, #0b2a62 66%, #071225 100%)`

### Fonts
Currently using Poppins from Google Fonts. Change in the `<head>` section of `portfolio.html`.

### Content
All content can be easily edited in the respective HTML sections.

## 🎯 Performance Optimizations

- Lazy loading of images with IntersectionObserver
- Efficient event delegation for animations
- CSS animations use `will-change` for smooth rendering
- Optimized scroll behavior with passive event listeners

## 🔐 Accessibility

- Semantic HTML5 structure
- ARIA labels for interactive elements
- Keyboard navigation support
- Focus-visible styles for keyboard users
- High contrast color scheme
