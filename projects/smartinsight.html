Overview

SmartInsight is an intelligent dashboard generation platform designed to simplify data analysis. Users can upload datasets and instantly receive visual reports, charts, graphs, and business insights.

The project focuses on transforming raw information into meaningful analytics without requiring technical expertise.

Tech Stack

Python Flask Pandas Matplotlib Plotly HTML CSS JavaScript Artificial Intelligence
