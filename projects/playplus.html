Play+ Video Streaming Platform

A full-stack video streaming platform inspired by YouTube, built using Java Spring Boot, React, JWT Authentication, and MySQL.

Live Demo Frontend Code Backend Code

Overview

Play+ is a modern video streaming platform that allows users to upload videos, watch content, manage channels, subscribe to creators, and maintain personalized profiles.

The project was built as a complete full-stack application using Java Spring Boot for the backend and React for the frontend.
