Overview

The E-Commerce Web Application is a full-stack online shopping platform developed using Java, JSP, Servlets, JDBC, and MySQL. It enables users to browse products, search items, manage shopping carts, and place orders while providing sellers with dedicated tools to manage their products.

The application follows the MVC architecture and offers secure session-based authentication, Cloudinary image uploads, responsive pages, and complete CRUD functionality for products, making it a scalable and practical web application.

Tech Stack

Java JSP Servlets JDBC MySQL Apache Tomcat HTML5 CSS3 JavaScript Cloudinary

Key Features

User Authentication

Secure user registration and login using session management.

Product Catalog

Browse and search products with detailed information.

Shopping Cart

Add, update, and remove products before checkout.

Seller Dashboard

Manage products with complete CRUD operations.

Image Upload

Upload and manage product images using Cloudinary.

Order Management

Track purchases and maintain customer order history.

Step 2 Browse Products
